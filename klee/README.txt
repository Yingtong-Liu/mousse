// KLEE Symbolic Virtual Machine
//===----------------------------------------------------------------------===//

KLEE is a symbolic virtual machine built on top of the LLVM compiler
infrastructure.

This version is built on top of the KLEE component of S2E. 
The original version (SHA1 ID: c3da567f87f15381c389c200be1181e603e84144) can be found on https://github.com/S2E/klee.git.
